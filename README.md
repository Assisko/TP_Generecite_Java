# TP - Généricité en Java

## Description
Ce projet illustre l'utilisation de :
- Héritage et classes abstraites (`Media`, `Livre`, `CD`)
- Interface (`Empruntable`)
- Polymorphisme et méthodes génériques (`GenericUtils`)
- Collections (`List`, `Set`, `Map`)
- Comparateurs personnalisés (`MediaComparator`, `LivreComparator`)

Il propose un menu interactif pour gérer les emprunts de médias et appliquer les différentes fonctionnalités du TP.
